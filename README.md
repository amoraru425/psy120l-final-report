# psy120l-final-report
Everyday Moments and The Mind

**Effects of Outdoor Exposure on Working Memory**

This repository contains the published HTML version of my PSY 120L final reproducible report. This project examined whether a brief outdoor break improves working memory performance compared to a brief indoor break, and whether the effect differed based on biological sex or perceived stress levels. The report was created using Quarto and R and the published report file is index.html. The dataset is not included in this repository.
